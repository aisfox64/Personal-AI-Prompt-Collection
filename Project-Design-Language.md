# 现代暗黑霓虹设计语言规范 (Modern Dark Neon)

## 设计哲学：极简未来与高对比沉浸
**核心理念**：抛弃繁杂的装饰，回归极致的深邃暗黑背景，通过高饱和度的**霓虹绿 (Neon Green)** 进行视觉引导。融合**高通透毛玻璃 (Glassmorphism 2.0)** 与**环境光 (Ambient Lighting)**，营造出一种悬浮、通透且充满生命力的未来科技感。

### 设计原则
1.  **极致暗黑 (Deep & Pure)**：背景不再是偏色的蓝黑，而是纯净的深灰与黑，提供无限的深度感。
2.  **霓虹点缀 (Neon Accent)**：使用极高饱和度的酸性绿作为核心交互色，在暗黑中形成强烈对比。
3.  **悬浮与通透 (Floating & Lucid)**：核心容器脱离文档流，采用大圆角悬浮设计，配合高斯模糊与饱和度提升，模拟高级光学玻璃质感。
4.  **环境光场 (Ambient Glow)**：界面不再是死寂的平面，而是通过背景中流动的微光营造呼吸感。

## 基础规范

### 色彩系统 (Color System)

#### 基础色阶
- `--deep-space: #161618` - 深空灰黑（主背景，去除蓝调，更纯粹）
- `--terminal-gray: #1C1C1E` - 终端灰（二级背景）
- `--interface-slate: #2C2C2E` - 界面岩灰（悬浮层背景）
- `--border-midnight: #3A3A3C` - 边界灰（深色边框）
- `--text-pure: #FFFFFF` - 纯白（标题/高亮）
- `--text-secondary: #8E8E93` - 次级灰（正文/说明，Apple系灰色）

#### 品牌与强调色
- `--brand-amber: #9DFD00` - **霓虹绿 (Neon Green)**：核心主色，用于按钮、激活状态、光标。
- `--neon-cyan: #D4FF80` - **青柠光 (Light Lime)**：辅助高亮，用于渐变末端、次级光晕。
- `--hover-glow: rgba(157, 253, 0, 0.3)` - 交互光晕。

#### 功能语义色
- `--status-critical: #FF453A` - 错误/警示 (Red)
- `--status-success: #9DFD00` - 成功 (与主色一致)
- `--status-progress: #0A84FF` - 进程/信息 (Blue)
- `--status-inactive: #636366` - 未激活

### 字体系统 (Typography)
- **主字体**: `"Inter", -apple-system, BlinkMacSystemFont, sans-serif` - 现代、几何、屏幕阅读优化。
- **代码/数据**: `"JetBrains Mono", monospace` - 开发者美学。

### 圆角与间距 (Radius & Spacing)
- **大圆角主导**：
    - 卡片：`24px`
    - 侧边栏/Dock：`24px` / `999px` (胶囊形)
    - 按钮：`999px` (全圆角)
- **间距**：宽松，强调呼吸感。

### 质感与特效 (Visual Effects)

#### Glassmorphism 2.0 (新一代毛玻璃)
不再是简单的白色半透明，而是强调**模糊**与**饱和度**的结合，配合微弱的边框光。
```css
.glass-panel {
  background: rgba(28, 28, 30, 0.4); /* 低不透明度 */
  backdrop-filter: blur(24px) saturate(180%); /* 高模糊 + 饱和度提升 */
  border: 1px solid rgba(255, 255, 255, 0.08); /* 极细微边框 */
  box-shadow: 
    0 4px 24px -1px rgba(0, 0, 0, 0.2),
    inset 0 0 0 1px rgba(255, 255, 255, 0.05); /* 内发光提升质感 */
}
```

#### 环境光 (Ambient Light)
在页面背景中使用径向渐变模拟光源，打破死黑。
- 顶部/局部：`rgba(157, 253, 0, 0.08)` (霓虹绿微光)

## 布局系统 (Layout)

### 悬浮导航架构 (Floating Navigation)
摒弃传统的贴边侧边栏，采用**悬浮式**设计。

1.  **桌面端 (Desktop)**
    - 形态：左侧悬浮胶囊面板。
    - 位置：`top: 16px`, `left: 16px`, `bottom: 16px`。
    - 交互：默认收起仅显示图标，悬停时展开显示详情。
    - 视觉：强毛玻璃质感，与背景分离。

2.  **移动端 (Mobile)**
    - 形态：底部悬浮 Dock (类似 iOS 底部条)。
    - 位置：`bottom: 24px`, `center`。
    - 内容：精简，仅保留核心导航与头像。
    - 交互：点击激活，激活项展开显示文字，未激活项仅显示图标。

## 组件规范

### 按钮 (Buttons)
- **Primary**: 
    - 背景：`#9DFD00` (Neon Green)
    - 文字：`#000000` (黑色，高对比度)
    - 阴影：`box-shadow: 0 0 16px rgba(157, 253, 0, 0.4)` (霓虹辉光)
- **Secondary**:
    - 背景：透明 / `rgba(255,255,255,0.1)`
    - 文字：`#FFFFFF`
    - 边框：`1px solid rgba(255,255,255,0.2)`

### 卡片 (Cards)
- 统一使用 `.glass-panel` 样式。
- 悬停效果：
    - 背景变亮：`rgba(28, 28, 30, 0.6)`
    - 边框发光：`border-color: rgba(157, 253, 0, 0.3)`
    - 阴影加深。

### 数据可视化 (Charts)
- 图表配色应严格遵循霓虹色系。
- 避免使用沉闷的深蓝或深红，优先使用亮色线条配合暗色填充区域。
- 雷达图/折线图：线条使用 `#9DFD00`，填充使用 `rgba(157, 253, 0, 0.2)`。

---

## 动效指南 (Motion)
- **曲线**：`cubic-bezier(0.2, 0.8, 0.2, 1)` - 快速响应，平滑回弹。
- **时长**：
    - 微交互：`200ms`
    - 展开/收起：`300ms` - `400ms`
- **反馈**：
    - 悬停：缩放 `scale(1.02)`，光晕增强。
    - 点击：缩放 `scale(0.95)`。

---

## 总结
这套设计语言旨在通过**极致的黑**衬托**极致的绿**，结合物理世界的光影质感（毛玻璃、环境光），打造一个既专业高效，又极具未来潮流感的界面。它不再是冰冷的机器终端，而是一个有温度、有呼吸的智能空间。
